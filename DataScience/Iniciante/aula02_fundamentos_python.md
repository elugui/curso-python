# Aula 02 — Fundamentos de Python

## Explicação Conceitual
- Sintaxe básica do Python: identação, comentários, comandos simples.
- Variáveis: declaração, tipos de dados (int, float, str, bool).
- Operadores: aritméticos, relacionais e lógicos.

## Prática com Exemplos
```python
# Exemplo de declaração de variáveis
idade = 25
nome = "Ana"
altura = 1.68

# Operadores
soma = 2 + 3
comparacao = idade > 18
logico = True and False
```

## Exercícios
1. Crie três variáveis: uma para idade, uma para nome e uma para altura.
2. Escreva um código que verifica se a idade é maior que 18.
3. Faça uma operação lógica usando duas variáveis booleanas.

## Resolução dos Exercícios
```python
# Exercício 1
idade = 30
nome = "Carlos"
altura = 1.75

# Exercício 2
if idade > 18:
    print("Maior de idade")
else:
    print("Menor de idade")

# Exercício 3
verdadeiro = True
falso = False
resultado = verdadeiro or falso
print(resultado)
```
