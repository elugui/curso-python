# Aula 05 — Introdução ao NumPy

## Explicação Conceitual
- O que é o NumPy e por que usar arrays?
- Diferenças entre listas e arrays.
- Operações básicas com arrays.

## Prática com Exemplos
```python
import numpy as np

# Criando um array
arr = np.array([1, 2, 3, 4])
print(arr)

# Operações
print(arr * 2)
print(arr + 10)
```

## Exercícios
1. Crie um array NumPy com 5 números inteiros.
2. Multiplique todos os elementos do array por 3.
3. Explique uma vantagem do uso de arrays NumPy em relação a listas.

## Resolução dos Exercícios
```python
import numpy as np
# Exercício 1
arr = np.array([10, 20, 30, 40, 50])

# Exercício 2
print(arr * 3)

# Exercício 3
# Arrays NumPy são mais rápidos e eficientes para operações matemáticas.
```
