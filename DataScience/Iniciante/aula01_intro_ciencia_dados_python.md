# Aula 01 — Introdução à Ciência de Dados e Python

## Explicação Conceitual
- O que é Data Science? Definição, aplicações e importância no mercado atual.
- Principais áreas de atuação e exemplos reais de uso.
- Visão geral do ciclo de vida de um projeto de Data Science.

## Prática com Exemplos
- Pesquisa de aplicações de Data Science em diferentes setores (saúde, finanças, tecnologia).
- Demonstração: Navegação em portais de vagas para identificar demandas de Data Science.

## Exercícios
1. Pesquise e escreva um breve resumo sobre uma aplicação de Data Science que você considera interessante.
2. Liste três áreas do mercado que utilizam Data Science e explique como.

## Resolução dos Exercícios
1. Resposta exemplo: "Na saúde, Data Science é usada para prever surtos de doenças analisando grandes volumes de dados de pacientes."
2. Resposta exemplo:
   - Saúde: Previsão de epidemias, diagnósticos automatizados.
   - Finanças: Detecção de fraudes, análise de risco de crédito.
   - Varejo: Recomendação de produtos, análise de comportamento do consumidor.
