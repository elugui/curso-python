# Aula 03 — Estruturas de Dados em Python

## Explicação Conceitual
- Listas, tuplas, dicionários e conjuntos: definição, diferenças e usos.
- Operações básicas: criação, acesso, modificação e remoção de elementos.

## Prática com Exemplos
```python
# Lista
frutas = ["maçã", "banana", "laranja"]
frutas.append("uva")

# Tupla
coordenadas = (10, 20)

# Dicionário
aluno = {"nome": "João", "idade": 20}

# Conjunto
numeros = {1, 2, 3, 2}
```

## Exercícios
1. Crie uma lista com 5 nomes e adicione um novo nome.
2. Crie um dicionário com informações de um livro (título, autor, ano).
3. Explique a diferença entre lista e tupla.

## Resolução dos Exercícios
```python
# Exercício 1
nomes = ["Ana", "Bruno", "Carlos", "Diana", "Eduardo"]
nomes.append("Fernanda")

# Exercício 2
livro = {"título": "1984", "autor": "George Orwell", "ano": 1949}

# Exercício 3
# Lista é mutável, tupla é imutável.
```
