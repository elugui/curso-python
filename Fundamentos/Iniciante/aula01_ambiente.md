# Aula 01 — Ambiente de Desenvolvimento

## Introdução Conceitual
Python é uma linguagem multiplataforma e pode ser utilizada em diversos sistemas operacionais. Para programar de forma produtiva, é importante configurar um ambiente de desenvolvimento eficiente. O VSCode é um editor moderno, gratuito e extensível, ideal para iniciantes e profissionais.

## Prática Guiada
### 1. Instalando o Python
- Acesse https://python.org e baixe a versão mais recente do Python 3 para seu sistema.
- Durante a instalação, marque a opção "Add Python to PATH".

### 2. Instalando o VSCode
- Baixe o VSCode em https://code.visualstudio.com/.
- Instale as extensões: Python (Microsoft), Pylance, Code Runner.

### 3. Configurando o VSCode
- Abra o VSCode, pressione Ctrl+Shift+P e selecione "Python: Select Interpreter" para escolher o Python instalado.
- Use o terminal integrado (Ctrl+`) para executar comandos Python.

### 4. Primeiro Script
```python
print("Hello, World!")
```
Salve como `hello.py` e execute no terminal:
```
python hello.py
```

## Exercício Proposto
1. Instale o Python e o VSCode.
2. Crie um script chamado `meu_nome.py` que exiba seu nome e profissão.

## Resolução Comentada
```python
# meu_nome.py
print("Meu nome é Ana e sou desenvolvedora de software.")
```

## Dicas VSCode
- Use o terminal integrado para rodar scripts rapidamente.
- Explore o atalho Ctrl+S para salvar arquivos.
