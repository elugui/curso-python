# Aula 01: Introdução à Análise de Dados e Ambiente Python

## Objetivos
- Compreender o que é análise de dados.
- Conhecer o ciclo de vida da análise de dados.
- Instalar Python, Jupyter e bibliotecas essenciais.

## Explicação Conceitual
A análise de dados é o processo de inspecionar, limpar e modelar dados para descobrir informações úteis e apoiar a tomada de decisão. O ciclo de vida da análise de dados inclui:
- Coleta de dados
- Limpeza e preparação
- Análise e exploração
- Visualização
- Comunicação dos resultados

Ferramentas essenciais: Python, Jupyter Notebook, pandas, numpy, matplotlib.

## Prática com Exemplos
1. Instalação do Python (https://www.python.org/downloads/)
2. Instalação do Jupyter Notebook:
   ```bash
   pip install notebook
   ```
3. Instalação das bibliotecas:
   ```bash
   pip install pandas numpy matplotlib
   ```
4. Executando o primeiro notebook:
   - Abra o terminal e digite:
     ```bash
     jupyter notebook
     ```
   - Crie um novo notebook e execute:
     ```python
     print("Olá, análise de dados!")
     ```
